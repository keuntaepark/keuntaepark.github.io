Overview

I built a web tool that predicts and visualizes wildfire spread directions using real-time wind data. Users can click on a map to see probabilistic spread predictions (N/E/S/W) rendered interactively.

  • Realtime Prediction: Lightweight algorithm using current wind direction and speed.
  • Visual & Interactive: Spread areas rendered directly on the map (Leaflet.js).
  • Scalable Architecture: Designed for future enhancements with historical fire and terrain data.

Methodology & Details

  • Spatial Prediction: Random Forest, XGBoost for handling terrain, vegetation, wind features.
  • Temporal Modeling: ARIMA, LSTM to capture seasonality, lag effects, sequential trends.
  • Optimization: Hyperparameter tuning balancing accuracy and computational cost.
  • Integration: Real-time + historical data pipelines generating clear, interpretable risk outputs.

개요

실시간 바람 데이터를 활용해 산불 확산 방향을 예측하고 시각화하는 웹 도구를 개발했습니다. 사용자가 지도를 클릭하면 동서남북 방향별 확산 확률이 인터랙티브하게 표시됩니다.

  • 실시간 예측: 현재 바람 방향과 풍속을 사용하는 가벼운 알고리즘.
  • 직관적 시각화: Leaflet.js로 지도에 직접 확산 영역을 표현.
  • 확장 가능성: 산불 이력 및 지형 데이터로 고도화할 준비 완료.

방법론 및 세부 내용

  • 공간 예측: Random Forest, XGBoost로 지형, 식생, 바람 등 혼합 피처 처리.
  • 시간 예측: ARIMA, LSTM으로 계절성, 지연 효과, 순차 패턴 반영.
  • 최적화: 정확도와 연산 비용 균형 맞추는 하이퍼파라미터 튜닝.
  • 통합: 실시간·과거 데이터 파이프라인 결합으로 명확하고 해석 가능한 위험 예측 제공.
