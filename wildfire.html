Problem

Conventional wildfire forecasts are critically flawed as they fail to quantify uncertainty, preventing decision-makers from gauging prediction reliability. They also fail to dynamically fuse stable historical models with volatile real-time data, or model how the fire's scale changes with current conditions. This results in an incomplete and untrustworthy picture for field teams.

Action

  • Redesigned the Prediction Target: Redefined the prediction goal from simple probabilities to an 8-directional vector of (probability, uncertainty) pairs, quantitatively presenting the confidence of each forecast.
  • Implemented Dynamic Alpha Algorithm: Discarded a fixed-weight approach. Introduced `calculateDynamicAlpha`, a function that calculates a dynamic weight based on real-time wind speed and model uncertainty, automatically balancing trust between historical and real-time data.
  • Context-Aware Spread Modeling: Developed `calculateDynamicMaxD` to dynamically compute the maximum spread distance (MaxD) based on real-time wind speed and humidity, enabling more realistic simulations.
  • Advanced Data Processing: Implemented an enhanced interpolation model (`interpolateProb`) that calculates a final uncertainty score for any arbitrary point by assessing the uncertainty of the four nearest data points.
  • Architected for Collaboration: Standardized all prediction outputs to the `predicted_spread.json` format and decoupled the API/Model/UI architecture to improve team collaboration and maintenance efficiency.

Result

  • Confidence-Based Decision Support: Provided uncertainty metrics (Â±%) with every prediction, empowering commanders to make more informed and responsible decisions based on the data's confidence level.
  • Improved Situational Accuracy: The dynamic weighting and spread distance calculations lead to predictions that react more accurately to changing weather conditions, such as wind gusts or dry spells.
  • Nationwide Predictive Coverage: The advanced interpolation model reliably provides predictions with quantifiable confidence levels even for areas without training data.
  • Secured Scalability & Improvement Path: The decoupled architecture facilitates easy integration of future variables (e.g., terrain), while a SHAP-based analysis provides a clear routine for continuous model improvement.

Business/Public Sector Relevance

  • Designed for adoption by wildfire response agencies with fine-tuned data granularity and responsive UX.
  • Planned integration with Korea Meteorological Administration and Forestry APIs for field deployment.
  • Scalable as a public safety tool or climate-tech startup foundation.
