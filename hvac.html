Project Overview

As project manager and data lead, I drove the LG Electronics HVAC Virtual Experience (Janâ€“Apr 2023) by aligning design, engineering, and external partners. The goal: deliver a high-performance virtual showroom for B2B engagement, supported by data-driven UX improvements and cross-functional alignment.

Key Challenges

  • Identified heavy Unity assets as the main bottleneck using walkthrough tests, developer logs, and user complaints.
  • Discovered missing user behavior data, making UX improvement decisions speculative.
  • Mediated design vs. engineering conflicts over visual fidelity vs. performance priorities.

Approach & Strategy

  • Led heatmap analysis in Python to detect low-traffic 3D zones; recommended asset downgrades to improve load speed.
  • Acted as cross-team facilitator, ensuring clear priorities and translating client goals into technical roadmaps.
  • Built a live Miro dashboard to track feedback, align progress, and keep stakeholders synchronized.

Key Outcomes & Business Impact

  • +100% YoY visitor growth, contributing to increased B2B leads and stronger sales pipeline.
  • +15% longer session times, reflecting better user engagement and navigation.
  • 20% faster load times, boosting user retention and experience quality.

What I Learned
